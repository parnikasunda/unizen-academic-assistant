# UniZen

UniZen is a source-aware academic assistant designed to help college students manage studies and access verified academic information in a calm and structured way.

## Project Overview
UniZen answers student queries using Large Language Models supported by a vector-based knowledge store built from official academic documents such as syllabus, lecture notes, and college guidelines.

## Technologies Used
- Large Language Models (LLM)
- Vector Databases
- DevOps (GitHub for version control)

## Key Features
- College-specific academic question answering
- Source-aware responses using document retrieval
- Reduced incorrect responses through vector-based similarity search
