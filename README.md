# UniZen

UniZen is a source-aware academic assistant designed to help college students manage studies and access verified academic information in a calm and structured way.

## Project Overview
UniZen answers student queries using Large Language Models supported by a vector-based knowledge store built from academic documents such as syllabus, lecture notes, and course materials.

## Technologies Used
- Large Language Models (LLM)
- Vector Databases (conceptual implementation)
- DevOps (GitHub for version control)

## Key Features
- College-specific academic question answering
- Source-aware responses using document retrieval
- Reduced incorrect responses through similarity-based matching
- Interactive user interface using Streamlit

## Workflow
1. Academic documents are processed and divided into smaller chunks  
2. Chunks are stored in a structured format  
3. User queries are matched with relevant content  
4. The system generates clear answers with source reference  

## Demo
The application provides a simple Streamlit interface where users can ask questions and receive answers based on NLP course documents.
